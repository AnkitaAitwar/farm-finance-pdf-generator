# Farm Finance PDF Report Generator

This project is a backend web application that accepts farm finance details through a web-based form and generates a downloadable PDF report.

---

## Features
- Farmer and crop detail form
- Multiple income and expense entries
- Automatic profit/loss calculation
- PDF generation with tables and chart

---

## Libraries Used
- Flask
- ReportLab
- Matplotlib
